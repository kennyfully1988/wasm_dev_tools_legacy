# WebAssembly Developer Tools (Legacy)

[Live Demo](https://kennyfully1988.github.io/wasm_dev_tools_legacy/)

## Description / 描述
This repository contains a self-contained WebAssembly developer utility designed primarily for converting image files into optimized text and data segments for WebAssembly (specifically for the Chicken Engine). It runs entirely in the browser with no external dependencies.
本仓库包含一个自包含的 WebAssembly 开发者工具，主要设计用于将图像文件转换为 WebAssembly 优化文本和数据段（专为小鸡引擎定制）。它完全在浏览器中运行，无需任何外部依赖。

## Project Status / 项目状态
Please note that the codebase is quite messy and unstructured. The developer has no plans to refactor or make any future updates to this code.
请注意，本项目的代码库非常混乱且缺乏结构。开发者未来不打算重构或对其进行任何更新。

## Features / 功能特性

    Convert images into format-compliant WebAssembly data string segments.

    支持将图像转换为符合格式要求的 WebAssembly 数据字符串段。

    Built-in font rendering using the custom embedded 'Exo' typeface.

    内置字体渲染，采用自定义嵌入式 'Exo' 字体。

    Multiple selector modes to alternate processing logic based on target assets.

    多种选择器模式，可根据目标资源交替处理逻辑。

    Zero configuration setup: single standalone HTML file that works offline.

    零配置安装：单个独立的 HTML 文件，支持离线工作。

    Convenient one-click copy and export mechanisms for converted assets.

    便捷的一键复制与导出机制，轻松获取转换后的资源。

## How to Use / 如何使用

    Open the wasm_dev_tools.html file in any modern web browser.

    在任何现代网页浏览器中直接打开 wasm_dev_tools.html 文件。

    Select your processing mode from the dropdown menu.

    从下拉菜单中选择您需要的工作模式。

    Click on the file picker to upload your target asset or image.

    点击文件选择器上传您的目标资源或图像。

    Review the generated data string in the output text area, then copy or export it.

    在输出文本区域中查看生成的代码段，然后进行复制或导出。
